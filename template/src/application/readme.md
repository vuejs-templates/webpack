# 目录说明
## main 存放投保流程相关模块
- adduser: 录入基本信息
- makeplan: 制定保险计划
- viewplan: 预览计划书
- addinfo: 完善信息
- notify: 健康告知
- viewinsurerpact: 预览投保书
- progressprocess: 投保进度
- orderlist: 保单查询列表
- orderdetail: 保单详情
- queryorder: 查询投保单

## base 存放投保主流程无关，或纯展示类型单页
- gallery: 页面大全
- appfeature: 产品特点展示页
- user: 用户中心相关，目前只有名片
